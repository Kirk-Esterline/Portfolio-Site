Striper-Gear

Working with Striper-Gear, a local Cape Cod Based, fishing lure company, we rebuilt and relaunched their website. One of the primary aims of the website is to provide information to Striper-Gear's distributors, as well as the final consumer. This required a shift in thinking about how information is presented, and what information is important. Working closely with the owner of Striper-Gear, we were able to develop a layout which provides value to all users of the website.

  • Styled the website to match the companies branding
  • Created custom photo galleries to display each fishing lure
  • Provided photo editing to better display each lure
  • Developed and wrote custom code to provide a seamless experience for mobile users
  • Integrated media content from multiple sources to best highlight Striper-Gear's lures
